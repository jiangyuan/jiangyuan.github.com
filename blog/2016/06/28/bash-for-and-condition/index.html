条件语句和循环可以统称为流程控制，是一门语言最基础的部分。
bash 的流程控制和大家熟悉的语言非常类似，所以这块上手应该很快。

case

case 其实就是我们熟悉的那个 swich ，但语法形式上有很大的不同。

  • 双引号包裹变量，这不是必须的
  • 每一个 Test 语句，必须以 ) 结尾
  • 每一个条件区块，必须以 ;; 结尾
  • 整个 case 区块，必须以 esac 结尾——esac case spelled backwards
