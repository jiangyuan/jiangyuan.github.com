玩转 Mac

vim

好吧，一直都觉得 vim 是用来炫技的，也因此没啥好感，直到上手了 mac 。

在一个很少接触命令行的系统里面，vim 真的体现不出什么价值，gvim 和其他优秀的编辑器相比也并没有什么突出。

但是，如果在一个频繁接触 terminal ，甚至大部分时间都和其打交道的系统里，vim 显得如此特别，因为你巴不得多有的工作都在 terminal 里面搞定。

vim 是为 terminal 而生的，想要体会他的与众不同，请先换系统。
